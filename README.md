# Blood-donation-system

The Blood Donation System is a web-based application designed to connect blood donors with recipients efficiently.  
 
🎯 Objectives

Maintain a centralized database of blood donors

Allow users to search donors by blood group and location

Enable donors to register and update their availability

🛠️ Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: Java (Servlet/JSP) / PHP 

Database: MySQL

Server: Apache Tomcat / XAMPP

🧪 How to Run the Project

For PHP:

Install XAMPP/WAMP

Place project in htdocs

Import database in phpMyAdmin

Run:

http://localhost/project-folder

🗃️ Database Structure (Sample Tables)
Donor Table

name
